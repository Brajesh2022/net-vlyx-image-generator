How This Project Was Made

This project was built with a "single-file" approach, meaning all the necessary code (HTML, CSS, and JavaScript) is contained within one index.html file. This makes it extremely easy to host and share.

1. Core Technologies

  • HTML: Provides the fundamental structure of the web page.
  • Tailwind CSS: A utility-first CSS framework used for all styling, loaded via a <script> tag.
  • JavaScript (JS): The "brain" of the application. It handles user interactions, API communication, and tab switching.

2. The API: Pollinations AI

We used the free and public API provided by Pollinations AI.

  • API Endpoint: https://image.pollinations.ai/prompt/{YOUR_PROMPT}
  • How it works: You send a text prompt as part of the URL. The API's server processes this prompt and sends back an image file.
  • Parameters: We add ?seed={RANDOM_NUMBER} for uniqueness and &width=512&height=512 to request a square image.

3. Understanding the JavaScript

The JavaScript logic is split into two main parts:

A. Tab-Switching Logic

This small script handles clicking the "Generator" and "Documentation" tabs. It adds/removes Tailwind classes to show the correct content div and highlight the active tab button.

B. Image Generator Logic

  • DOM Selection: Gets references to all the buttons, inputs, and image tags.
  • Event Listeners: Listens for clicks on the "Generate" and "Download" buttons.
  • generateImage() Function: Validates the prompt, shows the loader, builds the API URL, and sets the src of the image tag to fetch the new image.
  • downloadImage() Function: Uses fetch() to get the image data as a blob, creates a temporary local URL for it, and then programmatically "clicks" a hidden link to trigger the browser's download prompt.
  • imageEl.onload / imageEl.onerror: These events handle the success or failure of the image load, hiding the loader and showing either the image or an error message.

How Can Someone Make This?

  1. Create a new file named index.html.
  2. Copy and paste the entire code from this project into your file.
  3. Save and open the file in any web browser to run it.
  4. To share it, upload this single index.html file to a free hosting service like GitHub Pages.
